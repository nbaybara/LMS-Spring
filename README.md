This is a library management system project. I developed a web application by using Spring Boot.
Tech Stack

Spring Boot
Spring Data Jpa
Spring Security
Thymelaf
Bootstrap
Requirements
JDK 1.8 or newer
-Maven 3
-Lombok
-Thymeleaf Security
PORT
http://localhost:8081

RUN
mvn clean install && mvn --projects backend spring-boot:run
